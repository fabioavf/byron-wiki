# A comunicação como uma _skill_ treinável

De acordo com Viana (2007), a comunicação pode determinar o sucesso ou insucesso de um projeto, por isso, uma das bases para um projeto com êxito é a comunicação facilitada e a colaboração entre a equipe.

<image src={pinguim} alt='Ele podia ter explicado a situação melhor.' width={350} />

Claro que existem pessoas com mais facilidade em se comunicar, assim como existem pessoas com mais desenvoltura em esportes e etc. Diante disso, é preciso se esforçar com treino e prática para obter uma **boa comunicação**. E como *"treinar comunicação"*?
