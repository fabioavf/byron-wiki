# Problema 2

## Devs desmotivados, se comunicando pouco e entregando tarefas mal feitas.

Se os devs demonstrarem desmotivação no projeto, falta de comunicação e entrega mal feita das tarefas, minha abordagem inicial seria fazer reuniões individuais com cada um deles para compreender as razões por trás desse desânimo. Para resolver o problema eu tomaria providências como:

- Se necessário, realocaria as funções de cada membro da equipe para que todos se sintam mais engajados com suas respectivas tarefas e se identifiquem melhor com o trabalho que estão realizando, assim fazendo com que se sintam mais próximos do projeto e se animem para a execução das tarefas.
- Se os problemas forem de natureza pessoal, além de oferecer apoio emocional, buscaria redistribuir parte das responsabilidades desse dev para outros membros da equipe, aliviando sua carga de trabalho e permitindo com que ele tenha mais tempo para lidar com seus problemas pessoais. Em casos extremos, poderia considerar a substituição desse membro da equipe por alguém que não fazia parte do projeto.
- Se a causa da desmotivação for sobrecarga de trabalho, exploraria várias soluções, como recrutar mais desenvolvedores, negociar prazos estendidos com o cliente ou terceirizar partes adicionais da produção. Analisaria cada projeto individualmente para determinar a solução mais viável em cada caso.
