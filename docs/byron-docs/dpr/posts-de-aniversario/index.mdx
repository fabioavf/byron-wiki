# Processos Posts de aniversários

## Documento com aniversários dos membros
É realizado um [documento](https://docs.google.com/document/d/1qEUZl4unHLuX8NUb1jpSAVTQ1upEuwah6uSzDB0abyU/edit?usp=sharing) com todas as datas de aniversários dos membros. Nos casos dos trainees, os membros de DPR entram em contato com eles para solicitarem a data de aniversário deles ao final de cada processo seletivo. Esse documento já está pronto e fica em DPR confidencial. Depois disso o documento é enviado no grupo de whatsapp da byron para que os membros confira suas datas de aniversário, para organizar a conferência é feita uma listinha para que todos preencham quando já revisaram. Depois de todos os membros revisarem, esse mesmo membro de DPR coloca no cronograma do Notion de DPR as datas de cada aniversário dos membros.

**Quem**: Uma pessoa qualquer de DPR

**Quando**: É realizado no começo do ano.

**Para que**: Para coletar as informações e ter certeza que todos os aniversários serão coletados.

## Preparação para o aniversário
Na semana anterior ao aniversário de alguém, que está definido no cronograma, um membro de DPR fica responsável de entrar em contato com a diretoria do aniversariante para pedir características dessa pessoa. Para que com base nessa diretoria seja realizado um texto de aniversário próprio para essa pessoa.

**Quem**: Uma pessoa qualquer de DPR.

**Quando**: Uma semana antes do aniversário de algum membro.

**Para que**: Para fazer o textinho próprio de cada membro, tornando cada texto único.

## Post no Instagram 
O mesmo membro responsável pela etapa 3 fica responsável por inserir o texto no template, localizado na pasta de DPR confidencial, e postar no instagram da byron.

**Quem**: Uma pessoa qualquer de DPR.

**Quando**: No dia do aniversário do membro.

**Para que**: Para desejar felicitações ao membro aniversariante.
