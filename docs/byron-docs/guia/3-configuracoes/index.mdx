# Configurações do documento

Um recurso um pouco mais exclusivo do Docusaurus é a seção de opções do documento ao início de todo arquivo Markdown.

```md
---
title: 'Título da página'
sidebar_label: 'Título da página na sidebar'
---

Seu markdown aqui.
```

Basicamente, são uma série de metadados contendo informações sobre o próprio documento.

:::danger ATENÇÃO
Essas configurações devem, **SEMPRE**, ser a primeira coisa no documento. Caso contrário, não vão ser aplicadas e você vai ficar lelé da cuca tentando entender o que rolou.
:::
