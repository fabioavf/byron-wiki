# Markdown extendido

Nessa página, serão abordados os recursos novos que são específicos ao Docusaurus e que você pode usar em suas documentações.

## Admonições

Um novo recurso é o de admonições (sim, o nome é bizarro), que são "avisos" customizáveis. Usá-los é muito simples:

```md
:::note

Um pouco de **conteúdo** em _Markdown_ com `sintaxe` de admonições.

:::

:::tip

Um pouco de **conteúdo** em _Markdown_ com `sintaxe` de admonições.

:::

:::info

Um pouco de **conteúdo** em _Markdown_ com `sintaxe` de admonições.

:::

:::caution

Um pouco de **conteúdo** em _Markdown_ com `sintaxe` de admonições.

:::

:::danger

Um pouco de **conteúdo** em _Markdown_ com `sintaxe` de admonições.

:::
```

Note que os nomes estão todos em inglês, mas não se preocupe: podemos trocar os nomes! Para customizar o nome de qualquer admonição, basta colocar o nome personalizado depois de sua tag:

```md
:::tip dica

Vá tomar água. Faz bem. :)

:::
```
