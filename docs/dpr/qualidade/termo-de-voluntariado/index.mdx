Um termo de voluntariado é um documento ou acordo que estabelece os termos e condições nos quais uma pessoa se compromete a realizar trabalho voluntário em uma organização, como uma empresa ou uma entidade sem fins lucrativos.

## Etapas do processo

### Entrar na pasta

** Como: ** O primeiro passo é chegar até a pasta onde se encontram os termos de voluntariado. Para isso se segue o seguinte caminho:

:::info Pastas
DPR -> Administrativo -> Selo EJ -> Ano correspondente -> 1 fase -> Termos de voluntariado.
:::

### Escolher o template

** Como: ** Salvo que seja necessário editar um termo já existente, o próximo passo é utilizar o template que se encontra na pasta e fazer uma cópia dele.

### Preencher os campos

** Como **: O seguinte é mudar os campos (os quais vão estar marcados em azul) e colocar as informações correspondentes da pessoa da qual está sendo feito o termo de voluntariado.

### Mudar nomes

** Como: ** Caso seja necessário, tem que se mudar os nomes tanto das testemunhas como do presidente da Byron. 

### Assinar

** Como: ** O último ponto a ser feito é enviar para o respectivo membro o termo de voluntariado, para que assim ele possa assiná-lo. Para isso é usado um software de assinatura digital (DocuSign ou Zapsign por exemplo).
