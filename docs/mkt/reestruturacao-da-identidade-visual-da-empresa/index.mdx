## Etapas do processo

:::caution cuidado

Este processo é um compilado de várias páginas e  vários cursos até a execução final dele, portanto só siga para o passo de execução após verificar se já possui o conhecimento em cada curso apresentado.

### O que é identidade visual?

**Quem:** Membro de DC.

**Por que:**  Obter conhecimento sobre identidade visual.

**Como:** Acessar o link a seguir [identidade visual](https://smartalk.com.br/blog/identidade-visual/), para entender o que é identidade visual.

### Diferenças entre identidade visual, marca e branding

**Quem:** Membro de DC.

**Quando:** Assim que realizar o primeiro passo.

**Por que:** Entender a diferença entre estes três conceitos.

**Como:** Acessar o link a seguir [diferenças entre identidade visual, marca e branding](https://smartalk.com.br/blog/manual-de-identidade-visual/), e fazer a leitura para entender a diferença entre identidade visual, marca e branding.

**Quando:** Assim que realizar o segundo passo.

**Como:** Acessar o link a seguir [o que é posicionamento da marca](https://neilpatel.com/br/blog/posicionamento-de-marca/), e fazer a leitura para entender o que é posicionamento da marca.
