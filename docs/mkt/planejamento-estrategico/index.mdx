## Etapas do processo

### Buscar ideias

**Quem:** Qualquer membro da Diretoria Comercial - Comunica.

**Quando:** Uma vez a cada duas semanas.

**Por que:** Para buscar novas ideias de postagens, garantindo duas semanas de conteúdos para o instagram.

**Como:** O membro fica responsável por buscar ideias para novas postagens, garantindo no mínimo, duas semanas com conteúdos no instagram, atribuindo qual a cor que deverá ser a capa (azul ou branca) de cada post e um responsável para a criação.

Além disso, o membro responsável por coletar as ideias deverá colocá-las no Kanban - Status. Esse local mostra a situação de todas as publicações, mas o que é relevante é a distribuição de ideias no Status ideias.

Inicialmente nenhum membro é responsável nem há data para publicação, ela serve como um banco de conteúdos para que possa ser utilizado nas atualizações de postagens, evitando deixar uma semana sem atividade.

Assim como na imagem abaixo, há 15 ideias que serão usadas nas próximas semanas, apenas como reserva de conteúdo.

<image src={buscar_ideias} alt="Quadro Kanban mostrando ideias para posts" hideLegend />

### Atribuir postagens

**Quem:** Qualquer membro de Comunica

**Quando:** Uma vez a cada duas semanas.

**Por que:** Para organizar as postagens na ordem em que serão postadas.

**Como:** O membro deverá atribuir as informações às postagens, para isso ele deverá clicar no conteúdo desejado, em seguida irá aparecer uma janela com o título da postagem, onde deverá ser preenchida:

- Data: Datas posteriores, preenchendo o conteúdo da semana seguinte.
- Status: 
  - Ideia: banco de ideias; 
  - Programado: atribuído; 
  - Fazendo: em andamento; 
  - Feito: publicada;
- Responsável: Membros de comunica;
- Cor de Fundo: Seguindo o Xadrez das postagens, Azul e Branco. Não é necessário para Stories.

<image src={atribuir_postagem} alt="Quadro Kanban mostrando onde se atribui a função para um membro" hideLegend />

Por fim, basta voltar para o calendário e caso precise alterar novamente a data, arraste o card.
