### Introdução

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem padrão usada para criar páginas da web. HTML é composto por uma série de elementos que são usados para estruturar o conteúdo de uma página da web. Cada elemento é representado por uma "tag" e pode conter texto, imagens, links e outros tipos de mídia. Os navegadores da web interpretam o HTML para exibir o conteúdo de uma página da web aos usuários.
