**HTML** é uma sigla em inglês que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). Ela serve para organizar e dar sentido as informações que você manda para o navegador. De uma maneira geral, fazendo uma correlação com um jornal, por exemplo, seria as informações que contém dentro desse jornal e a organização do que cada coisa representa.

Para que isso seja feito, o HTML é composto por uma série de elementos que são usados para estruturar o conteúdo de uma página da web. Cada elemento é representado por uma "tag" e pode conter texto, imagens, links e outros tipos de mídia.

Existem dois tipos de tags:

- **Block-level** (nível de bloco), que ocupa toda a largura de seu elemento pai, que também chamamos de elemento container, criando assim um “bloco”.
- **Inline**, geralmente usamos para demarcação de conteúdos de texto.
