Agora vou te mostrar outro tipo de caracterização das tags, a semântica.

## Tags semânticas

As tags **semânticas** são aquelas que dão sentido e significado para a informação que você quer passar. Voltando ao exemplo do jornal, seriam tags que definiriam o que é uma sessão, o rodapé, o cabeçalho, um artigo, uma lista, entre outros.

Algumas das principais tags utilizadas são:

- [`<header>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/header)
- [`<main>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/main)
- [`<footer>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/footer)
- [`<section>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/section)
- [`<article>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/article)
- [`<aside>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/aside)
- [`<nav>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/nav)
- [`<ol>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/ol)
- [`<ul>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/ul)
- [`<li>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/li)

## Tags sem semântica

As tags **não semânticas** são aquelas que não definem um significado para aquela informação, geralmente são usados para organização de tags em divisões ou para estilização.

Algumas das principais tags utilizadas são:

- [`<div>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/div)
- [`<span>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/span)
- [`<b>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/b)
- [`<i>`](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/i)
