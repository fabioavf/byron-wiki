# Se habituando com o Github

<video src="https://youtube.com/embed/BYo46nbEj8U" />

Nesse vídeo eu te mostro:

- Brevemente o que é SSH
- Como criar um par de chaves SSH e adicionar a chave pública ao Github
- Como clonar um repositório e como "subir" um repositório local pra um
  repositório remoto
- Alguns novos comandos do git
  - `git clone`
  - `git push`
  - `git pull`
  - `git remote add`
  - `git remote -v`
  - Entre outros que você pode conferir em alguma das folhas de dicas linkadas
    no índice da capacitação aqui na byron.wiki
- Como criar uma issue e para o que elas servem
- Como criar um pull request
- Como resolver conflitos de merge
- Como pesquisar na documentação do Git por comandos que você tem dúvida sobre
  como funcionam
- E, por fim, como ver o primeiro commit do Git, feito pelo Linus Torvalds em
  2005 (como prometi que faria na primeira aula)
