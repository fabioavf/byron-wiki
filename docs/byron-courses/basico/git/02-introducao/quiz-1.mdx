# Quiz 1

As questões abaixo fazem menção ao conteúdo da primeira aula em texto da
capacitação.

Responda sem consultar o material.

## Perguntas

**1. Qual a principal função de um sistema de versionamento de código?**

- [ ] Auxiliar na colaboração entre desenvolvedores
- [ ] Fazer rollbacks para versões antigas do código
- [ ] Manter um histórico de alterações na codebase
- [ ] Salvar o código na nuvem
- [ ] Fazer a release do software nas lojas de aplicativos

**2. O que motivou a criação do Git?**

- [ ] O BitKeeper se tornou software proprietário
- [ ] O BitKeeper era lento
- [ ] Linus Torvalds desejava desenvolver um sistema como o Subversion
- [ ] Os desenvolvedores controlavam as versões do código usando uma hierarquia
      de pastas
- [ ] Torvalds queria competir com o Mercurial

**3. O que NÃO é uma característica do Git?**

- [ ] É um sistema distribuído
- [ ] Permite a colaboração por meio de desenvolvimento não linear
- [ ] Possui várias ferramentas
- [ ] É rápido, porém inseguro
- [ ] É uma ferramenta de controle de versão
