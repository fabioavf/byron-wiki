Você já aprendeu o que são os arrays là na aula de variáveis, mas ainda não sabe como manipulá-los, o que é de extrema importância e utilidade. A partir de agora você verá como fazer isso.

## O que é?

A função [`map()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map) é um método que cria um novo array populado por elementos que surgem a partir de uma função que envolve cada elemento de outro array. Ou seja, é um método que pega cada elemento de um array, invoca uma função chamada `callback` com ele, por exemplo uma operação matemática, e retorna um novo array com o resultado de cada uma dessas funções.

```js
let Dinheiro = [2, 10, 20, 50, 100]; // Array de números

// Varre cada elemento do array "Dinheiro"
let Tigrinho = Dinheiro.map((element) => {
  return element / 2; // Pega cada um desses elementos e divide por 2
});

console.log(Dinheiro); // => [2, 10, 20, 50, 100]
console.log(Tigrinho); // => [1, 5, 10, 25, 50]
```

## Sintaxe

Como já visto anteriormente a estrutura de um `map()` consiste em:

```js
var novoArray = array.map(callback);
// ou
var novoArray = array.map(callback, thisArg);
```

Mas o que significa `callback` e `thisArg`? Eles são os parâmetros que o método recebe, sendo eles:

- **callback**: Função vista anteriormente, cujo retorno produz o elemento do novo Array. Recebe três argumentos:
  - **element**: O valor do elemento original do Array de origem.
  - **index**: O índice do elemento atual que está sendo processado no array.
  - **array**: O Array de origem.
- **thisArg** (opcional): Valor a ser utilizado como o this no momento da execução da função callback.

## Exemplos

Agora vamos ver alguns exemplos de aplicação do método `map()`, **tente fazer antes de ver a resposta!**

- Mapeando um array de números para um array de raízes quadradas.

```js
var num = [1, 4, 9];
var raiz = num.map(Math.sqrt); // Calcula a raiz quadrada em todos os elementos dos array

console.log(num); //  => [1, 4, 9]
console.log(raiz); // => [1, 2, 3]
```

- Mapeando um array de números para um array com o dobro do valor dos elementos.

```js
var num = [1, 4, 9];
var dobro = num.map((element) {
  return element * 2; // Pega cada um desses elementos e multiplica por 2
});

console.log(num); //   => [1, 4, 9]
console.log(dobro); // => [2, 8, 18]
```
