Para finalizar arrays, veremos o método `reduce()`, porém existem mais inúmeros métodos relacionados ao array, como o [`sort()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort), o [`push()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/push), o [`pop()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/pop), entre outros. Não deixe de continuar pesquisando para aprofundar seus conhecimentos!

## O que é?

O método [`reduce()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/reduce) executa uma função para cada elemento do array e retorna um valor único. Geralmente esta função envolve o valor atual do array e o anterior, possibilitando, por exemplo, o uso de operações matemáticas e comparação entre esses dois elementos.

## Sintaxe

Apesar da estrutura deste método ser semelhante aos outros, aqui temos algumas mudanças significativas: os parâmetros recebidos na função `callback` e o parâmetro de `valorInicial`.

```js
array.reduce(callback(acumulador, valorAtual, index, array), valorInicial);
```

- **callback**: Função que é executada em cada valor no array (exceto no primeiro, se nenhum valorInicial for passado); recebe quatro argumentos:
  - **acumulador**: É o valor que o callback anterior retornou. Este valor inicia com o valorInicial e será retornado na última iteração.
  - **valorAtual**: É o valor do elemento atual.
  - **index**: O índice do elemento atual que está sendo processado no array.
  - **array**: O Array de origem.
- **valorInicial** (opcional): Valor a ser utilizado como o acumulador na primeira chamada da função callback. Caso o valorInicial não seja fornecido, a primeira função de callback toma o primeiro elemento como o acumulador e o valorAtual não é lido.

## Exemplos

Agora vamos ver alguns exemplos de aplicação do método `reduce()`, **tente fazer antes de ver a resposta!**

- Calcular a soma dos elementos de um array.

```js
var num = [1, 2, 3, 4, 5]; // Array de números

var sum = num.reduce(function (acumulador, atual) {
  return acumulador + atual; // Pega o acumulador e soma o valor do elemento atual
}, 0); // Estabelece o valor inicial como 0

console.log(sum); // => 15
```

- Calcular o produto dos elementos de um array.

```js
var num = [1, 2, 3, 4, 5]; // Array de números

var product = num.reduce(function (acumulador, atual) {
  return acumulador * atual; // Pega o acumulador e multiplica o valor do elemento atual
}, 1); // Estabelece o valor inicial como 1

console.log(product); // => 120
```

- Retornar o maior número de um array.

```js
var num = [1, 2, 3, 4, 5, 0]; // Array de números

var max = num.reduce((acumulador, atual) => {
  return acumulador > atual ? acumulador : atual; // Compara o acumulador com o elemento atual e retorna o maior deles
});

console.log(max); // => 5
```
