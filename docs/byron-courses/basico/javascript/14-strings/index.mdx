Você já aprendeu o que são os strings lá na aula de variáveis, mas ainda não sabe como manipulá-las, o que é de extrema importância e utilidade. A partir de agora você verá como fazer isso.

## Concatenação de strings

Existem algumas formas de juntar strings durante o código no JavaScript, uma dessas maneiras é “somando” elas, por exemplo:

```js
var nome = "Livianu";
var idade = 19;

var frase = "Meu nome é " + nome + " e tenho " + idade + " anos";

console.log(frase); // => Meu nome é Livianu e tenho 19 anos
```

Como podem ver, este método é funcional, podemos concatenar até strings com números, porém não é o melhor método. O código fica confuso, grande e é visualmente feio. Então é mais comum vermos desenvolvedores declarando strings por meio de crases para facilitar a concatenação de strings com outras informações, já que isso permite inserir informações utilizando `${}`, dessa forma:

```js
var frase = `Meu nome é ${nome} e tenho ${idade} anos.`;

console.log(frase); // => Meu nome é Livianu e tenho 19 anos.
```

:::caution
Se atente em usar as **crases**, não as apas simples, nem aspas duplas e nem qualquer tipo de símbolo parecido! É um erro comum no começo e que pode te causar raiva desnecessariamente.
:::
