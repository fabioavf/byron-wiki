Agora vamos aprender dois métodos semelhantes, o `every()` e o `some()`.

## O que são?

- [**`every()`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/every): O método `every()` verifica se **todos** os elementos de um array seguem uma determinada condição estabelecida na função de callback, caso isso aconteça, retorna `true`, caso contrário retorna `false`.

- [**`some()`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/some): O método `some()` é bem semelhante, porém ele verifica apenas se **algum** elemento satisfaz a condição.

A sintaxe de ambos é igual às dos demais métodos e não precisa ser atribuída a alguma variável.

```js
array.every(callback);
// ou
array.some(callback);
```

## Exemplos

Agora vamos ver alguns exemplos de aplicação dos métodos `every()` e `some()`, **tente fazer antes de ver a resposta!**

- Verificar se **todos** os elementos de um array de números são ímpares.

```js
let num = [16, 24, 57, 73, 89, 95]; // Array de números

console.log(
  num.every((element) => {
    // Varre cada elemento do array
    return element % 2 != 0; // Verifica se é ímpar
  })
); // => false
```

- Verificar se **algum** dos elementos de um array de números é ímpar.

```js
let num = [16, 24, 57, 73, 89, 95]; // Array de números

let resposta = num.some((element) => {
  // Varre cada elemento do array
  return element % 2 != 0; // Verifica se é ímpar
});
console.log(resposta); // => true
```
