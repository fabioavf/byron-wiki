Agora vamos aprender dois métodos semelhantes, o `find()` e o `findIndex()`.

## O que são?

- [**`find()`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find): O método `find()` verifica o **primeiro elemento** de um array que satisfaz uma determinada condição estabelecida na função de callback e se caso isso aconteça, retorna o **valor** deste elemento.
- [**`findIndex()`**](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/findIndex): O método `findIndex()` é bem semelhante, porém ele retorna o **index** do **primeiro elemento** que satisfaz a condição.

:::note
Caso nenhum elemento satisfaça a condição, retorna **undefined**.
:::

A sintaxe de ambos é igual às dos demais métodos e não precisa ser atribuída à alguma variável.

```js
array.find(callback);
// ou
array.findIndex(callback);
```

## Exemplos

Agora vamos ver alguns exemplos de aplicação dos métodos `find()` e `findIndex()`, **tente fazer antes de ver a resposta!**

- Retornar o valor do primeiro elemento ímpar de um array.

```js
let num = [16, 24, 57, 73, 89, 95]; // Array de números

console.log(
  num.find((element) => {
    // Varre cada elemento do array
    return element % 2 != 0; // Verifica se é ímpar
  })
); // => 57
```

- Retornar o index do primeiro elemento maior que 57 de um array.

```js
let num = [16, 24, 57, 73, 89, 95]; // Array de números

let resposta = num.findIndex((element) => {
  // Varre cada elemento do array
  return element > 57; // Verifica se é maior que 57
});

console.log(resposta); // => 3
```
