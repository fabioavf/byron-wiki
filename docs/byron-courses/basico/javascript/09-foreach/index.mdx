Agora que já aprendemos sobre o `map()`, vamos conhecer e entender um método bem parecido, o `forEach()`.

## O que é?

O método [`forEach()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach) executa uma função para cada elemento do array, esta função, ao contrário do `map()`, não precisa necessariamente retornar um novo valor, ou seja, ela só vai executar a função e não necessariamente vai substituir o valor do elemento.

```js
const bora = ["b", "y", "r", "o,", "n"];
bora.forEach((element) => console.log(element)); // Da um console.log com o conteúdo de cada elemento

// Expected output: "b"
// Expected output: "y"
// Expected output: "r"
// Expected output: "o"
// Expected output: "n"
```

## Sintaxe

A sintaxe do `forEach()` é igual à que vimos anteriormente em `map()`, recebendo os mesmos parâmetros inclusive na função callback, porém ela não está vinculada a um novo array e sim ao mesmo que você já havia criado.

```js
array.forEach(callback);
// ou
array.forEach(callback, thisArg);
```

## Exemplos

Agora vamos ver alguns exemplos de aplicação do método `map()`, **tente fazer antes de ver a resposta!**

- Contando ocorrências de um elemento em um array.

```js
const letras = ["a", "b", "a", "c", "a", "b"];
let contagemA = 0; // Contagem de "a"

letras.forEach((letra) => {
  // Se o conteúdo do elemento for "a"
  if (letra === "a") {
    contagemA++; // Adiciona 1 na contagem
  }
});

console.log(contagemA); // => 3
```

- Criando uma frase com todas as strings de um array.

```js
const palavras = ["Palmeiras", "não", "tem", "mundial"];
let frase = "";

palavras.forEach((palavra) => {
  frase += palavra + " "; // adiciona o conteúdo do elemento e um espaço
});

console.log(frase); // => "Palmeiras não tem mundial"
```
