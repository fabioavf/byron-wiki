# Ferramentas básicas

Aqui estão listadas algumas ferramentas que serão necessárias e/ou úteis durante as capacitações.

## VScode

Todo programador precisa de um bom editor de código. Aqui na byron, o que nós mais usamos e recomendamos é o Visual Studio Code, ou VScode para os íntimos. Ele é gratuito e possui uma série de funções que podem facilitar muito a sua vida.

Você pode fazer o download por meio deste [link](https://code.visualstudio.com/).

## Node

Grande parte do código que desenvolvemos na byron envolve JavaScript. Por isso, é extremamente importe que você possua o Node instalado na sua máquina. 

Para intalar, basta acessar [aqui](https://nodejs.org) e escolher a versão mais recomendada. Se você usa Windows, um executável irá ser baixado. Agora, é só executar e seguir os passos do instalador. Simples assim.

Para conferir o sucesso da instalação. Basta rodar o seguinte comando em um terminal a sua escolha:

```bash
node -v
```

Caso a versão apareça, tudo ocorreu bem, você já possui o Node instalado.

Se você é usuário de Mac ou Linux, [aqui](https://www.treinaweb.com.br/blog/instalacao-do-node-js-windows-mac-e-linux) está um tutorial de instalação para você seguir.
