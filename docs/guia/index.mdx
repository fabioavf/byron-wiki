# Guia de documentação

Salve, meu querido byroniano! Essa é a nossa tão querida byron.wiki começando a sair, e pra isso tô escrevendo esse arquivo de documentação pra servir de guia em qualquer dúvida que você possam vir a ter!

O Docusaurus é um gerador de documentação automatizado baseado em React (!!!) feito pelo Facebook (que também desenvolveu o React). Contudo, você não precisa saber React pra popular nossa querida documentação (ufa!). Basta saber Markdown e, no máximo, um HTML básico, coisa que todos os efetivados da byron sabem.

O Markdown usado é, na verdade, o MDX, que é uma extensão do Markdown comum para suportar recursos de linguagem do JSX (JS + HTML, usado em React). Na prática, você não precisa saber muito além de colocar a extensão `.mdx` nos arquivos de documentação.

Além disso, caso você tenha alguma dúvida que não foi respondida por esse guia, olhe a [documentação do Docusaurus](https://docusaurus.io/docs/).
