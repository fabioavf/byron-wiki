# Publicando alterações

:::caution wip

Essa página ainda está em progresso. Preciso conferir como será feita a publicação por não-administradores.

:::

Por fim, depois de você enriquecer nossa querida documentação com mais conhecimento que não será perdido, o último passo é de publicar todas as mudanças!

Para isso, você fará um `commit` de todas as alterações e um `push`. Feito isso, o mantenedor da Wiki lerá as suas alterações e, caso sejam aprovadas, será feito um merge à `main`. O site será recompilado automaticamente (viva a modernidade!).

```bash
git commit -m "descreva quais documentos foram alterados/inseridos"

git push
```
