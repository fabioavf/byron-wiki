# Localização dos documentos

Depois de ter esse repositório rodando certinho, você já pode começar a escrever os nossos documentos!

**Todos** os documentos vão ficar dentro da pasta `/docs`. A separação das páginas e o conteúdo na sidebar é feito automaticamente, de acordo com a própria estrutura de arquivos da documentação.

Basicamente, a estrutura vai ser `/docs/{diretoria}`. Então, se vamos ~~re~~mapear DPR, por exemplo, os processos vão ficar localizados em `/docs/dpr/`, certo?

Feito isso, você deve criar uma pasta para o próprio documento. Então, se você estiver mapeando o Planejamento Financeiro, por exemplo, crie a pasta `/docs/dpr/planejamento-financeiro`. Dentro dessa pasta, existirão:

- O arquivo `index.mdx`
- A pasta `img`

O arquivo `index.mdx` será onde você vai passar maior parte do seu tempo. É lá que acontece a mágica do Markdown e tudo que esperamos. Escreva Markdown comum como aprendeu nas capacitações (e [relembre](/wiki/guia/markdown-guide/) nessa seção), salvo algumas [exceções](/wiki/guia/mdx-guide/).

A pasta `img` será onde você vai salvar as imagens usadas nos processos.
